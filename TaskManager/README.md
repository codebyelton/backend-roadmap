# 📝 TaskManager

Um gerenciador de tarefas desenvolvido em **C#**, criado como parte da evolução dos meus estudos de programação.

O projeto será desenvolvido gradualmente, começando com funcionalidades simples e evoluindo conforme novos conceitos forem aprendidos.

## 🚀 Evolução do Projeto

* [ ] **Criar Tarefa**

  * Permitir que o usuário crie e registre uma nova tarefa.

* [ ] **Listar Tarefas**

  * Exibir todas as tarefas cadastradas no sistema.

* [ ] **Remover Tarefas**

  * Permitir que o usuário remova uma tarefa existente.

---

## 🎯 Objetivo

Praticar conceitos de programação utilizando C#, desenvolvendo o projeto de forma incremental e evoluindo sua estrutura conforme novos conhecimentos forem adquiridos.
